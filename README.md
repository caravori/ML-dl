# Requirements
- Linux (wget)
- JSON 
- Requests
- Python3
# Usage


```
python3 ml-dl.py 

```
Put your manga code, and that's it. 

The manga code is located in url eg. https://mangalivre.net/manga/ao-ashi/8412
8412 is your code.
